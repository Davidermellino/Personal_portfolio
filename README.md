# Data Science Portfolio

A professional React portfolio website designed for a Data Science student, showcasing skills, projects, and accomplishments in a modern, responsive interface.

## Features

- **Modern UI**: Clean, professional design with smooth animations
- **Responsive Layout**: Optimized for all devices (mobile, tablet, desktop)
- **Dark/Light Theme**: Toggle between themes with context-based state management
- **Interactive Sections**: Home, About, Projects, and Contact
- **Project Showcase**: Filterable project cards with data visualizations
- **Skills Display**: Visual representation of technical proficiencies
- **Education Timeline**: Interactive display of academic history
- **Contact Form**: Functional form with validation
- **Animations**: Smooth transitions using Framer Motion
- **Data Visualization**: Using Recharts for project metrics

## Technology Stack

- React 18
- React Router for navigation
- Bootstrap 5 for styling
- Framer Motion for animations
- Recharts for data visualization
- React Icons for icon components

## Installation

To get started with this project, follow these steps:

1. Clone the repository
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```

## Development

This project uses Vite as its build tool. Here are the available scripts:

- `npm run dev` - Starts the development server
- `npm run build` - Builds the app for production
- `npm run preview` - Previews the production build locally
- `npm run lint` - Lints the codebase

## Project Structure

```
/src
  /components      # Reusable UI components
  /pages           # Main page components
  /hooks           # Custom React hooks
  /context         # React context providers
  /assets          # Static assets
  /utils           # Utility functions
  /styles          # CSS styling files
```

## Customization

To personalize this portfolio:

1. Update personal information in the Home and About components
2. Replace project data in the Projects component
3. Modify the color scheme in the CSS variables
4. Update social media links and contact information
5. Replace the profile image with your own photo

## Deployment

To build the app for production:

```bash
npm run build
```

This will create a `dist` directory with the production-ready files.
